# Local-Gemma4-Deployment-Guide

一份基于 RTX 5060 + R9 硬件环境的本地 AI 大模型部署实战方案。

## 🎯 解决了什么问题？
- 完美解决 Ollama 默认 C 盘爆满问题。
- 解决下载大模型时的 `unexpected EOF` 网络中断问题。
- 提供全自动断点续传部署脚本。

## 🚀 快速开始
1. 确保安装了 [Ollama](https://ollama.com)。
2. 克隆本仓库或下载 `/scripts` 下的自动化脚本。
3. 运行脚本，一键拉取 Gemma 4 模型。
