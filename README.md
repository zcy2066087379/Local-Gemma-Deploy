本地 AI 部署指南，助力你的个人算力进化。

## 为什么选择本项目？
- **环境隔离：** 利用 D 盘仓库彻底解决空间管理难题。
- **一键挂机：** 自动处理网络波动，确保下载不中断。
- **普适性：** **本指南及自动化脚本不仅适用于 Gemma 4。只需修改脚本中的模型 ID，即可一键部署 Llama 3、Qwen 等任何开源模型。**

## 📥 快速开始
1. 确保已安装 [Ollama](https://ollama.com)。
2. 配置环境变量 `OLLAMA_MODELS` 指向 D 盘。
3. 运行 `/scripts` 下的自动化部署脚本。
